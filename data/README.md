# Dataset Information

The datasets used in this project is Brazilian E-Commerce Public Dataset by Olist from Kaggle.

Due to their large size, they are not hosted directly in this repository's main view but are required for the analysis.

## Source
- **Name**: Brazilian E-Commerce Public Dataset by Olist
- **Source**: [Olist Dataset - Kaggle](https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce)


## File Structure
- `olist_customers_dataset.csv`
- `olist_geolocation_dataset.csv`
- `olist_order_items_dataset.csv`
- `olist_order_payments_dataset.csv`
- `olist_order_reviews_dataset.csv`
- `olist_orders_dataset.csv`
- `olist_products_dataset.csv`
- `olist_sellers_dataset.csv`
- `product_category_name_translation.csv`


Please download these files and place them in this `data/` directory to run the notebook locally.
